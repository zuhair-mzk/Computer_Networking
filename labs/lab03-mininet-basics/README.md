# Lab 03: Mininet Basics

## Overview
Introduction to Mininet network emulation platform for creating and testing virtual networks.

## Learning Objectives
- Understanding Mininet architecture
- Creating virtual network topologies
- Running basic network tests
- Measuring network performance

## Files
- `LabReport03.pdf` - Lab report and findings

## Setup
```bash
# Install Mininet
sudo apt-get install mininet

# Start basic topology
sudo mn --topo single,3
```

## Experiments
1. Connectivity testing with ping
2. Bandwidth measurement with iperf
3. Custom topology creation
4. Performance analysis

## Tools Used
- Mininet
- iperf
- ping
- Wireshark
