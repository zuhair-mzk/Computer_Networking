# Course Resources

## Overview
This directory contains supplementary materials, slides, and reference documentation used throughout the CSCD58 course.

## Structure

### `slides/`
Contains tutorial presentations and lecture slides:
- Tutorial materials for labs
- Supplementary lecture content
- Step-by-step guides

### `reference-material/`
Additional reference documentation:
- Protocol specifications (RFCs)
- Tool documentation
- Research papers
- External tutorials

## Usage
These materials supplement the official course content and provide additional context for assignments and labs.
